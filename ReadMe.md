# Ví dụ hướng dẫn lập trình Express cùng với việc viết Unit Test

# Sử dụng SuperTest
[SuperTest](https://www.npmjs.com/package/supertest) được dùng để kiểm thử HTTP server trong đó có Express

Cài đặt
```bash
yarn add supertest -D
```
